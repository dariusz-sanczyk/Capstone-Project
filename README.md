# Best Shop - Capstone Project

Multi-page e-commerce website for suitcases, built with TypeScript, SASS, and Vite.

**Author:** Dariusz Sańczyk

## 📋 Prerequisites

- Node.js (v18 or higher recommended)
- npm (comes with Node.js)

## 🚀 Installation

```bash
# Clone the repository
git clone <your-repo-url>
cd fundamentals-project-ts-template

# Install dependencies
npm install
```

## 💻 Running the Project

### Development Mode
```bash
# Start development server with hot reload
npm run dev
```

The application will open automatically at `http://localhost:5173`

### Production Build
```bash
# Build for production
npm run build

# Preview production build
npm run preview
```

### Compile SASS Only
```bash
npm run compile
```

## 🧪 Code Quality

```bash
# Run all linters
npm run lint

# Lint TypeScript only
npm run lint:ts

# Lint SCSS only
npm run lint:scss
```
